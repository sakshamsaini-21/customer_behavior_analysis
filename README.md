# customer_behavior_analysis

# 📊 Data Analytics Project

## Overview

This project demonstrates an end-to-end data analytics workflow, starting from raw data analysis to business insights. It covers data loading, exploratory data analysis (EDA), data cleaning, SQL analysis, interactive dashboard creation in Power BI, report generation, and presentation preparation using Gamma.

The goal of this project is to showcase practical data analytics skills using industry-standard tools and techniques.

## Dataset

* Imported dataset into Python for analysis.

* Cleaned and prepared the dataset for further analysis.
* Used the processed data for SQL queries and dashboard development.

## Project Workflow

### 1. Data Loading

* Imported the dataset into Python.
* Performed initial inspection of rows, columns, and data types.

### 2. Exploratory Data Analysis (EDA)

* Analyzed data distributions.
* Identified missing values and duplicates.
* Explored relationships between variables.
* Generated visualizations to understand trends and patterns.

### 3. Data Cleaning

* Removed duplicate records.
* Handled missing values.
* Corrected inconsistent data.
* Prepared the dataset for analysis.

### 4. SQL Analysis

* Imported the cleaned dataset into a SQL database.
* Wrote SQL queries to answer business questions.
* Used filtering, grouping, joins, aggregate functions, and sorting for analysis.

### 5. Power BI Dashboard

* Built an interactive dashboard.
* Added KPIs and charts.
* Included filters and slicers for dynamic analysis.
* Presented key business insights through visualizations.

### 6. Report & Presentation

* Created a detailed project report summarizing findings.
* Developed a presentation using Gamma to communicate insights effectively.

## How to Run

1. Clone this repository.
2. Install the required Python libraries.
3. Open the Jupyter Notebook.
4. Run the notebook to perform EDA and data cleaning.
5. Import the cleaned dataset into PostgreSQL, MySQL, or SQL Server.
6. Execute the SQL queries provided in the `sql` folder.
7. Open the Power BI dashboard (`.pbix`) to explore the visualizations.
8. Review the project report and presentation for detailed insights.
